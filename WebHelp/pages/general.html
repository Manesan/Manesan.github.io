General

The Black Gold Project web application is an online system and requires no installation other than navigating to the following URL: https://manesan.github.io the development team has already deployed the web application onto a Microsoft Azure server. Once the website is loaded, the user will need to provide their login credentials to access to the system.

Please note that each user is assigned a certain access level according to the web application access will be restricted accordingly.
