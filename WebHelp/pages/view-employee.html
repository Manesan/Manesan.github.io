View employee

Description:

This is the View Employee Profile modal. On this modal the user can view an employee’s details and click the button to edit the employee’s details.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “View Employee Profile”.

2

Edit this employee button

A button to open the “Update Employee” modal for the user to update the specific employee’s details when clicked and closes the “View Employee Profile” modal.

3

Panel header

Panel header titled “Personal Details”. Can be collapsed or expanded to view and hide panel content.

4

Name label

Label titled “Name”.

5

Name textbox

Input of type Text with the employee’s Name. This is disabled as its populated with the employee’s Name.

Label titled “Surname”.

12

Email label

Label titled “Email”.

13

Email textbox

Input of type Text with the employee’s Email. This is disabled as its populated with the employee’s Email.

14

ID Number label

Label titled “ID Number”.

This is displayed if the user provided the employee’s id number else a label titled “Passport Number” will be displayed if the employee’s passport number was provided when adding the employee.

ID Number textbox

Input of type Text with the employee’s ID Number. This is disabled as its populated with the employee’s ID Number.

This textbox is displayed if the user provided the employee’s id number else the textbox Passport Number will be displayed with the employee’s passport number provided when the employee was added.

21

Employee Types label

Label titled “Employee Types”.

22

These are a list of labels with the different Employee Type Descriptions associated with the specific employee.

23

A button to delete the specific employee type. This is disabled and can only be clicked when the “Update Employee” modal after the “Edit this employee” button is clicked

28

Cancel button

A button that closes the modal when clicked.
