View rental agreement

Description:

This is the View Rental Agreement modal. On this modal the user can view the rental agreement details and click a button to accept/reject a rental agreement.

 

Reference

Control

Functionality description

1

Modal header

Label titled “View Rental Agreement”

2

Accept/reject button

A button that opens the “Accept/Reject” modal

3

Modal text

Modal text containing the property address for the rental agreement

4

Personal details label

Label titled “Personal Details”

5

Name textbox

This textbox is populated with the name of the client that is logged in

6

Surname textbox

This textbox is populated with the surname of the client that is logged in

7

Contact number textbox

This textbox is populated with the contact number of the client that is logged in

8

Email textbox

This textbox is populated with the email of the client that is logged in

9

Ok button

This button closes the modal when it is clicked.
