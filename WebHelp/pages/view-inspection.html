View Inspection

Description:

This is the View Inspection modal. On this modal the user can view an inspection for a property.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “View Inspection”

2

Edit this inspection button

A button that will allow the user to edit information of the inspection when clicked.

3

Modal text

Modal text containing the property address for the inspection

4

Download inspection document button

This button will download a PDF document for the inspection when clicked

5

Inspection date label

Label titled “Date”

6

Inspection Date textbox

This textbox is populated with the date of the inspection

7

Valuation description label

Label titled “Inspector”

Inspector textbox

9

Inspection type label

Label titled “Type”

10

Inspection type textbox

This textbox is populated with the inspection type of the inspection

11

Inspection status label

Label titled “Status”

12

Status textbox

This textbox is populated with the status of the inspection

13

Defects label

Label titled “Defects”

14

Comments label

Label titled “Comments”

15

Comments button

This text area is populated with the inspection comment of the inspection

16

Cancel button
