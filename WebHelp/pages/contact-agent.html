Contact agent

Description:

This screen allows a client to send a message to Black Gold Properties through the website. The client needs to enter the following their information. All the information will be sent to Black Gold Properties through email.

Reference

Control

Functionality description

1

Modal Header

Modal header titled “Contact Black Gold Properties”

2

Modal label

Label titled “Do you have any questions?”

3

First name textbox

Input of type Text to add the client Name. Enter the client’s first name in this field.

4

Last name textbox

Input of type Text to add the client Last Name. Enter the client’s last name in this field.

5

Email address textbox

Input of type Text to add the client Email address. Enter the client’s email in this field.

6

Country code dropdown

Input of type dropdown to select the country code for the client’s contact number. Click this to select client’s contact number country code.

7

Contact number textbox

Input of type Text to add the client Contact. Enter the client’s contact in this field.

8

Subject textbox

Input of type Text to add the client Subject. Enter the client’s message subject in this field.

9

Message text area

Input of type Text to add the client Message. Enter the client’s message in this field.

10

Send Message button

This button will send all the information provided as an email to Black Gold Properties.
