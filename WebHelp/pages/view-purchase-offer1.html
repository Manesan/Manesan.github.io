View Purchase Offer

 

Description:

This is the View Purchase Offer modal. On this modal a user can view all the purchase offer information. The user can also accept or reject the purchase offer by clicking buttons on the modal.

 

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “View Purchase Offer”

2

Accept button

This button opens the “Accept this purchase offer?” modal when clicked.

3

Reject button

This button opens the “Reject this purchase offer?” modal when clicked.

4

Address label

Label titled “Property Address”.

5

Address textbox

Textbox is populated with the property address

6

Offer date label

Label titled “Purchase Offer Date”.

7

Offer date textbox

Textbox is populated with the purchase offer date

8

Amount label

Label titled “Purchase Offer Amount(R)”.

9

Amount textbox

Textbox is populated with the purchase offer amount

10

Cancel button

This button closes the modal
