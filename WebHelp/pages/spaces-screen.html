Spaces screen

Description:

This is the Spaces screen. On this screen the user can search for a space by providing the space’s Description in the relevant search bar to view the space and maintain spaces. The user can also add a new space and delete an existing space. The screen displays all the spaces that currently exist on the system.

 

Reference

Control

Functionality description

1

Screen header

Screen header titled “Spaces” with a sub-header titled “View and maintain spaces”.

2

Add a space button

A button that opens the “Add Space” modal to add a new space.

3

Search bar

Input of type Text to search through the grid list cards.

4

Grid list card

A content container with the space description, a button to view the space and a button to delete the space of the specific card.

5

Card title

Card Title with the space’s Description.

6

View this space button

Button that opens the “View Space” modal to view the specific space’s details when clicked.

7

Delete this space button

Button that opens the “Are you sure you want to delete this space?” confirmation modal to delete the specific space when clicked.
