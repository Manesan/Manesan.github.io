How to activate or use the online help function

On the Black Gold Properties web application, the user has two ways of accessing the help function:

➢  The first way is by clicking the help function on the nav bar which will take you to the full user manual.

➢  Another option is clicking the help button on top of the search bar on each screen to take you to that specific section of the user manual.
