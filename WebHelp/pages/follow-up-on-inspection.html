Follow Up On Inspection

Description:

This is the Follow Up On Inspection modal. On this modal the user can follow up on an inspection that has already been captured for a property.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “Follow Up On Inspection”

2

Modal text

Modal text containing the property address for the inspection

3

Modal prompt

Label titled “Please provide the updated inspection details”. This is to prompt the user as to what is needed on this modal.

4

Inspection date label

Label titled “Date”

5

Inspection date date picker

This date picker displays the date at which the inspection was conducted

6

Inspector label

Label titled “Inspector”

7

Inspector textbox

This textbox is populated with the inspector name and surname of the inspection which is disabled and cannot be updated from this modal

8

Inspection type label

Label titled “Type”

9

Inspection type textbox

This textbox is populated with the type of the inspection

10

Inspection status label

Label titled “Status”

11

Inspection status textbox

This textbox is populated with the status of the inspection

12

Inspection defects label

Label titled “Defects”

13

Inspection defect description label

Label titled “Defect Description”

14

Inspection defect description dropdown

This dropdown is used to select the defect description

15

Inspection defect space description label

Label titled “Space Description”

16

Inspection defect space description dropdown

This dropdown is used to select the defect space description

17

Inspection defect quantity label

Label titled “Defect Quantity”

18

Inspection defect quantity numeric up down

This numeric up down is used to select the defect quantity

19

Add defect button

This button adds a defect to the inspection

20

Inspection added defect label

This label displays the defect description, space description and defect quantity of the existing (or newly added) defect

21

Delete this defect button

This button deletes the added defect

22

Inspection comments label

Label titled “Comments”

23

Inspection comments text area

This text area is populated with the comments for the inspection

24

Inspection document label

Label titled “Upload Inspection Document (.PDF)”

25

Choose file input

Input of type file to choose the document to upload. Click this to upload a new inspection document.

26

Save button

This button will close the modal window and open the “Save this inspection?” Confirmation modal.

9

Cancel button

This button will close the modal window when clicked and discard any changes made on this modal
