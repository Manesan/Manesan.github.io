Add Point of interest

Name:

This is the Add Point of interest modal. On this modal the user can add a point of interest by providing the point of interest’s details.

 

Reference

Control

Functionality name

1

Modal header

Modal header titled “Add Point of interest”.

2

Modal prompt

Label titled “Please provide the point of interest details”. This is to prompt the user to provide the point of interest details.

3

Name label

Label titled “Name”.

4

Name textbox

Input of type Text to add the point of interest Name. Enter the name in this field.

5

Type label

Label titled “Type”.

6

Type dropdown

Dropdown box populated with existing point of interest types.

7

Suburb label

Label titled “Suburb”.

8

Suburb dropdown

Dropdown box populated with existing suburbs

9

Add button

A button that will confirm the specific point of interest to be added by opening the “Add this point of interest?” Confirmation modal and closing the “Add Point of interest” modal when clicked.

10

Cancel button

A button that closes the modal when clicked.
