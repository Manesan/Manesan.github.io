Update Property Owner

Description:

This is the Update Property Owner modal. On this modal the user can update the property owner’s details except the ID number.

 

 

Reference

Control

Functionality description

1

Modal header

Header titled “Update Property Owner”

2

Modal prompt

Modal prompt with the text “Please provides the updated property owner details” that will prompt the user to know what to provide on this modal

3

Name label

Label titled “Name”.

4

Name textbox

Textbox is populated with the property owner’s Name. Enter updated name in this field.

5

Surname label

Label titled “Surname”.

6

Surname textbox

Textbox is populated with the property owner’s Surname. Enter updated surname in this field.

7

Email label

Label titled “Email”.

8

Email textbox

Textbox is populated with the property owner’s Email. Enter updated email in this field.

9

ID Number label

Label titled “ID Number”.

10

ID Number textbox

Textbox is populated with the property owner’s ID Number and disabled.

11

Address label

Label titled “Address”.

12

Address textbox

Textbox is populated with the property owner’s Address. Enter updated address in this field.

13

Contact number label

Label titled “Contact Number”

14

Contact textbox

Textbox is populated with the property owner’s Contact. Enter updated contact in this field.

15

Alternative contact number label

Label titled “Alt Contact Number”

16

Alternative contact textbox

Textbox is populated with the property owner’s Alternative Contact. Enter updated alternative contact in this field.

17

Update button

This button opens the “Update this property owner?” modal

18

Cancel button

This button closes the modal
