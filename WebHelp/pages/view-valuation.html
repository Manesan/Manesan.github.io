View Valuation

Description:

This is the View Valuation modal. On this modal the user can view a valuation for a property.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “View Valuation”

2

Edit this valuation button

A button that will allow the user to edit information of the valuation when clicked

3

Modal text

Modal text containing the property address for the valuation

4

Download valuation document button

This button will download a PDF document for the valuation when clicked

5

Valuation Date textbox

This textbox is populated with the date of the valuation

6

Valuation Description textbox

This textbox is populated with the description of the valuation

7

Valuer textbox

This textbox is populated with the name and surname of the valuation’s valuer

8

IV Status textbox

This textbox is populated with the status of the valuation

9

Cancel button

This button will close the modal window when clicked

10

Valuation date label

Label titled “Valuation Date”

11

Valuation description label

Label titled “Valuation Description”
