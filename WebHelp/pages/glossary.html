Term

Definition

Azure

Microsoft Azure is a cloud computing service created by Microsoft for building, testing, deploying and managing applications and services through a global network of Microsoft- managed data centres.

 

Cancel

Annul, revoke or drop.

 

Credentials

Identity information

 

Field

An area that requires an entry.

 

Link

An item that redirects you to the named screen.

 

Logout

Exiting and closing the software.

 

Modal

A dialog/form that appears on top of the main content screen.

 

Password

A secret word or phrase that must be used to gain admission to the system.

 

Redirect

Navigate to a new screen.

 

Toast

A popup alert that generally appears on the right-hand side of the screen and is usually for notifications with great importance.

 

Upload

To load a document onto the system.

 

User

Anyone using the web application or desktop application.

 

Web application

The online client and administration side of the system
