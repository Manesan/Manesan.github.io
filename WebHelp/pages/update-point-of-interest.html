Update Point of interest

Name:

This is the Update Point of interest modal. On this modal the user can update a point of interest’s name.

 

Reference

Control

Functionality name

1

Modal header

Modal header titled “Update Point of interest”.

2

Modal prompt

Label titled “Please provide the updated point of interest details”. This is to prompt the user to provide the updated point of interest name.

3

Name label

Label titled “Name”.

4

Name textbox

Input of type Text to update the point of interest Name. Enter the point of interest’s updated name in this field.

 

Type label

Label titled “Type”.

 

Type dropdown

Dropdown box populated with existing point of interest types.

5

Suburb label

Label titled “Suburb”.

6

Suburb dropdown

Dropdown box populated with existing property types.

7

Update button

A button that will confirm the specific point of interest to be updated by opening the “Update this point of interest?” Confirmation modal and closing the “Update Point of interest” modal when clicked.

8

Cancel button

A button that closes the modal when clicked.
