Reset password

Description:

This is the Reset Password screen. This screen allows a user to reset their password. If the user is not registered they can go to the sign-up page with a button.

Reference

Control

Functionality description

1

Modal header

Modal header titled “Reset Your Password”.

2

Email label

Label titled “Enter your email address”.

3

Password textbox

Input of type Text to enter email address. Enter your email address in this field

4

Description label

Label titled “Don’t have an account”. A link that opens the “Sign-up” screen.

6

Reset password button

A button that closes the modal and rests password when clicked.
