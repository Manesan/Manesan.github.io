Add province

Description:

This is the Add Province modal. On this modal the user can add a province by providing the province’s details.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “Add Province”.

2

Modal prompt

Label titled “Please provide the province details”. This is to prompt the user to provide the province details.

3

Name label

Label titled “Name”.

4

Name textbox

Input of type Text to add the province Name. Enter the province name in this field.

5

Add button

A button that will confirm the specific province to be added by opening the “Add this province?” Confirmation modal and closing the “Add Province” modal when clicked.

6

Cancel button

A button that closes the modal when clicked.
