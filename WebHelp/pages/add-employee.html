Add employee

Description:

This is the Add Employee modal. On this modal the user can add an employee by providing the employee’s details.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “Add Employee”.

2

Modal prompt

Label titled “Please provide the employee details”. This is to prompt the user to provide the employee details.

3

Panel header

Panel header titled “Personal Details”. Can be collapsed or expanded to view and hide panel content.

4

Name label

Label titled “Name”.

5

Name textbox

Input of type Text to add the employee’s Name. Enter the employee’s name in this field.

6

Surname label

Label titled “Surname”.

7

Surname textbox

Input of type Text to add the employee’s Surname. Enter the employee’s surname in this field.

8

Contact Number label

Label titled “Contact Number”.

9

Country code dropdown

Input of type dropdown to select the country code for the employee’s contact number. Click this to select employee’s contact number country code.

10

Contact Number textbox

Input of type Text to add the employee’s Contact Number. Enter the employee’s contact number in this field.

11

Alternative Contact Number label

Label titled “Alternative Contact Number”.

12

Country code dropdown

Input of type dropdown to select the country code for the employee’s alternative contact number. Click this to select employee’s alternative contact number country code.

13

Alternative Contact Number textbox

Input of type Text to add the employee’s Alternative Contact Number. Enter the employee’s alternative contact number in this field.

14

Email label

Label titled “Email”.

15

Email textbox

Input of type Text to add the employee’s Email. Enter the employee’s email in this field.

16

South African? Label

Label titled “South African?”.

17

South African? Radio button options

Input of type Radio to select if the employee is south African or not. Select yes if the user has an ID Number and no if the user is not south African.

 

When the user selects yes, a label titled “ID Number” will be displayed with an ID Number textbox of type Text. Enter the employee’s id number in the field.

 

When the user selects no, a label titled “Passport Number” will be displayed with a Passport Number textbox of type Text. Enter the employee’s passport number in this field.

 

18

Address label

Label titled “Address”.

19

Address textbox

Input of type Text to add the employee’s Address. Enter the employee’s address in this field.

20

Banking Details label

Label titled “Banking Details”.

21

Banking Details textbox

Input of type Text to add the employee’s Banking Details. Enter the employee’s banking details in this field.

22

Panel header

Panel header titled “Employment Details”. Can be collapsed or expanded to view and hide panel content.

23

Employee Types label

Label titled “Employee Types”.

24

Add employee type button

A button that will add the employee type selected from the Employee Types dropdown when clicked. A user can now select a new employee type if they wish to in the Employee Types dropdown.

25

Employee Types dropdown

Dropdown box populated with the existing Employee Type Descriptions, obtained from the Employee Type table.

26

Date Employed label

Label titled “Date Employed”.

27

Date Employed datepicker

Input of type Date to select the employee Employed Date. Click this to select the employee employed date.

28

Remuneration label

Label titled “Remuneration”.

29

Remuneration textbox

Input of type Text to add the employee’s Remuneration. Enter the employee’s remuneration in this field.

30

Password label

Label titled “Password”.

31

Password textbox

Input of type Text to add the employee’s Password. Enter the employee’s password in this field.

32

Add button

A button that will confirm the specific employee to be added by opening the “Add this employee?” Confirmation modal and closing the “Add Employee” modal when clicked.

33

Cancel button

A button that closes the modal when clicked.
