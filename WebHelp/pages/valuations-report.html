Valuations Report

Description:

This is the Valuations Report. This report displays a table with all the property valuations done for the report date range displaying the Valuation Date, Property Address, Valuer and Valuation Description for each valuation. The report has the organisation details, the report date and the details of the person who prepared the report.

 

Reference

Control

Download Report button

Report header titled “Valuations Report”.

Prepared By: label

Label titled “Prepared By:” followed by the agents name and surname.

6

Report Date Range table

A table that displays the report date range that was provided for in the Generate Valuation Report model.

7

Valuations Report table

A table that displays the property valuations for the report date range with the Valuation Date, Property Address of the property that was valued, the Valuer and the Valuation Description.
