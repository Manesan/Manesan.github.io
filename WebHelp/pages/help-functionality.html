Help functionality

The web application has help links all throughout the system which are used to help the user navigate through the system. This will guide the user should they get confused on how the system works and need help explanation.

The help button can be found on the nav bar:

Click this button whenever you need help, and the system will be redirect you to the help document where you may find further information on the screen and how to navigate other aspects of the system.
