Capture Valuation

Description:

This is the Capture Valuation modal. On this modal the user can edit a valuation for a property.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “Capture Valuation”

2

Modal text

Modal text containing the property address for the valuation

3

Valuation Date textbox

This textbox is populated with the date of the valuation which can be changed using the date picker icon on the far left of the textbox

4

Valuation Description textbox

This textbox is populated with the description of the valuation which can be changed by typing in the new description in this textbox

5

Valuer textbox

This textbox is populated with the valuer’s name and surname of the valuation which is disabled and cannot be updated from this modal

6

IV Status textbox

This textbox is populated with the status of the valuation

7

Choose file input

Input of type file to choose the document to upload. Click this to upload the valuation document.

8

Capture button

This button will close the modal window and open the “Capture this valuation?” Confirmation modal.

9

Cancel button

This button will close the modal window when clicked and discard any changes made on this modal

10

Modal prompt

Label titled “Please provide the updated valuation details”. This is to prompt the user as to what is needed on this modal.

11

Valuation date label

Label titled “Valuation Date”

12

Valuation description label

Label titled “Valuation Description”

13

Valuer label

Label titled “Valuer”

14

Valuation status label

Label titled “IV Status”

15

Valuation document label

Label titled “Valuation Document (.PDF)”
