Reports screen

Description:

This is the Reports screen. On this screen the user can generate system reports and search for the different system reports by the report name in the relevant search bar.

 

Reference

Control

Functionality description

1

Screen header

Screen header titled “Reports” with a sub-header titled “Generate system reports”.

2

Search bar

Input of type Text to search through the grid list cards.

3

Grid list card

A content container with the report name and the button to generate the reports.

4

Generate a report of all system documents button

A button that generates the audit report and displays the screen with the generated audit report.

5

Card title

Card Title with the report name.

6

Generate an agent report button

A button that generates the agent report and displays the screen with the generated agent report.
