Inspection Report

Description:

This is the Inspection Report. This report displays a table with all the property valuations done for the report date range displaying the Valuation Date, Property Address, Valuer and Valuation Description for each valuation.

 

The report has the organisation details, the report date and the details of the person who prepared the report.

 

Reference

Control

Functionality description

1

Download Report button

A button that downloads the Inspection Report in PDF format.

2

Organisation information header

Organisation header with the company name, contact number, email address and the company logo.

3

Report header

Report header titled “Inspection Report”.

4

Report Date: label

Label titled “Report Date:” followed by the date the report was generated

5

Prepared By: label

Label titled “Prepared By:” followed by the agents name and surname.

6

Report Date Range table

A table that displays the report date range that was provided for in the Generate Inspection Report model.

7

Take-on Inspection label

Label titled “Take-on Inspections” to indicate the table that follows.

8

Take-on Inspections table

A table that displays the take-on inspections for the report date range with the Inspection Date, Property Address for each property, the Inspector and any recorded Defects.

The table also has a count for the total number of take-on inspections for the report date range.

9

Outgoing Inspection label

Label titled “Outgoing Inspections” to indicate the table that follows.

10

Outgoing Inspections table

A table that displays the outgoing inspections for the report date range with the Inspection Date, Property Address for each property, the Inspector and any recorded Defects.

The table also has a count for the total number of outgoing inspections and a total number of inspections for the report date range.

11

Number of Inspections per Inspection Type pie chart

A pie chart that display the number of inspections per inspection types in the inspection report.
