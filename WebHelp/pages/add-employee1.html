Add employee

Description:

This is the Add this employee? modal. On this modal the user can confirm whether or not they want to add the employee.

 

Reference

Control

Functionality description

1

Confirmation modal header

Modal header titled “Add this employee?”.

2

Confirmation modal content

Label with the name and surname of the employee to be added.

3

Yes button

A button that adds the specific employee when clicked.

4

No button

A button that closes the confirmation modal when clicked.
