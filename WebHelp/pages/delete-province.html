Delete province

Description:

This is the Are you sure you want to delete this province? modal. On this modal the user can confirm whether or not they want to delete the province.

 

Reference

Control

Functionality description

1

Confirmation modal header

Modal header titled “Are you sure you want to delete this province?”.

2

Confirmation modal content

Label with the province name to be deleted.

3

Yes button

A button that updates the specific province’s name when clicked.

4

No button

A button that closes the confirmation modal when clicked.
