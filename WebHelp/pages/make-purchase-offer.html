Make purchase offer

 

Description:

This is the Make a Purchase Offer modal. This modal allows the client to make a purchase offer for a specific property. The client must upload some documents before submitting.

 

Reference

Control

Functionality description

1

Modal header

Label titled “View Rental Agreement”

2

Bank statement label

Label titled “Bank statement of the last 3 months from today(.PDF)”

3

Choose file input

Input of type file to choose the document to upload. Click this button to upload a bank statement document.

4

ID/Passport label

Label titled “Certified copy of ID/Passport(.PDF)”

5

Choose file input

Input of type file to choose the document to upload. Click this button to upload a certified copy document.

6

Offer amount label

Label titled “Offer Amount ( R)”

7

Offer amount textbox

Input of type Text to add the offer amount. Enter the offer amount in this field.

8

Submit button

This button opens the “Submit purchase offer?” modal

9

Cancel Button

This button closes the modal
