View defect

Description:

This is the View Defect modal. On this modal the user can view a defect’s details and click the button to edit the defect description.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “View Defect”.

2

Edit this defect button

A button to open the “Update Defect” modal for the user to update the specific defect description when clicked and closes the “View Defect” modal.

3

Description label

Label titled “Description”.

4

Description textbox

Input of type Text with the defect Description. This is disabled as its populated with the defect Description.

5

Cancel button

A button that closes the modal when clicked.
