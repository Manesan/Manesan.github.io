Cities screen

Description:

This is the Cities screen. On this screen the user can search for a city by providing the city’s Name in the relevant search bar to view the city and maintain cities. The user can also add a new city and delete an existing city. The screen displays all the cities that currently exist on the system.

 

Reference

Control

Functionality description

1

Screen header

Screen header titled “Cities” with a sub-header titled “View and maintain clients”.

2

Add a city

A button that opens the “Add City” modal to add a new city.

3

Search bar

Input of type Text to search through the grid list cards.

4

Grid list card

A content container with the name of the city, a button to view the city and a button to delete the city of the specific card.

5

Card title

Card Title with the city’s Name.

6

View this city button

Button that opens the “View City” modal to view the specific city’s details when clicked.

7

Delete this city button

Button that opens the “Are you sure you want to delete this city?” confirmation modal to delete the specific city when clicked.
