Update employee

Description:

This is the Update Employee modal. On this modal the user can update an employee’s details and delete any employee types they want to delete as well.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “Update Employee”.

2

Modal prompt

Label titled “Please provide the updated employee details”. This is to prompt the user to provide the updated employee details.

3

Panel header

Panel header titled “Personal Details”. Can be collapsed or expanded to view and hide panel content.

4

Name label

Label titled “Name”.

5

Name textbox

Input of type Text to update the employee’s Name. Enter the employee’s updated name in this field.

6

Surname label

Label titled “Surname”.

7

Surname textbox

Input of type Text to update the employee’s Surname. Enter the employee’s updated surname in this field.

Contact Number label

Label titled “Contact Number”.

9

Contact Number textbox

Input of type Text to update the employee’s Contact Number. Enter the employee’s updated contact number in this field.

10

Alternative Contact Number label

Label titled “Alternative Contact Number”.

11

Alternative Contact Number textbox

Input of type Text to update the employee’s Alternative Contact Number. Enter the employee’s updated alternative contact number in this field.

12

Email label

Label titled “Email”.

13

Email textbox

Input of type Text to update the employee’s Email. Enter the employee’s updated email in this field.

14

ID Number label

Label titled “ID Number”.

This is displayed if the user provided the employee’s id number else a label titled “Passport Number” will be displayed if the employee’s passport number was provided when adding the employee.

15

ID Number textbox

Input of type Text to update the employee’s ID Number. Enter the employee’s updated id number in this field.

This is displayed if the user provided the employee’s id number when adding the employee else the Passport Number textbox will be displayed. The user will provide the employee’s updated passport number.

16

Address label

Label titled “Address”.

Address textbox

Banking Details label

Label titled “Banking Details”.

Banking Details textbox

Input of type Text to update the employee’s Banking Details. Enter the employee’s updated banking details in this field.

20

Panel header

Panel header titled “Employment Details”. Can be collapsed or expanded to view and hide panel content.

21

Employee Types label

Label titled “Employee Types”.

22

Add employee type

A button that will add the employee type selected from the Employee Types dropdown when clicked. A user can now select a new employee type if they wish to in the Employee Types dropdown.

23

Employee Types dropdown

Dropdown box populated with the existing Employee Type Descriptions, obtained from the Employee Type table.

24

Delete employee type

A button to delete the specific employee type selected from the dropdown and added using the “Add employee type” button.

25

Employee Types content

These are a list of labels with the different Employee Type Descriptions associated with the specific employee.

26

Date Employed label

Label titled “Date Employed”.

27

Date Employed textbox

Input of type Date to update the employee’s Date Employed. Select the employee’s updated date employed in this field.

28

Remuneration label

Label titled “Remuneration”.

29

Remuneration textbox

Input of type Text to update the employee’s Remuneration. Enter the employee’s updated remuneration in this field.

30

Update button

A button that will confirm the specific employee to be updated by opening the “Update this employee?” Confirmation modal and closing the “Update Employee” modal when clicked.

31

Cancel button

A button that closes the modal when clicked.
