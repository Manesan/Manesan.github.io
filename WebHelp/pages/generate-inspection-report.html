Generate Inspection Report

Description:

This is the Generate Inspection Report modal that allows the user to enter date parameters to that will generate an inspection report on the system.

Reference

Control

Functionality description

1

Modal title

Modal titled “Generate Inspection Report”

2

Modal prompt

Modal prompt with the text “Please provide the inspection report criteria” that will prompt the user to know what to provide on this modal

3

From date label

Label titled “From Date”

4

From date date textbox

Date textbox where the user can use the date picker to select the starting date for the report

5

To date label

Label titled “To Date”

6

To date date textbox

Date textbox where the user can use the date picker to select the ending date for the report

7

Generate Report button

Clicking this button will close the modal and open a new browser window with the generated report

8

Cancel button

Clicking this button will close the modal, discarding any changes made and return the user to the “Inspection” screen
