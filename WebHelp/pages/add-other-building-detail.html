Add other building detail

Description:

This is the Add Other Building Detail modal. On this modal the user can add a other building detail by providing the other building detail’s details.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “Add Other Building Detail”.

2

Modal prompt

Label titled “Please provide the other building detail details”. This is to prompt the user to provide the other building detail details.

3

Description label

Label titled “Description”.

4

Description textbox

Input of type Text to add the other building detail Description. Enter the description in this field.

5

Add button

A button that will confirm the specific other building detail to be added by opening the “Add this other building detail?” Confirmation modal and closing the “Add Other Building Detail” modal when clicked.

6

Cancel button

A button that closes the modal when clicked.
