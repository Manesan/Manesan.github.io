Upload credit check report

Description:

This is the Upload Credit Check Report modal. On this modal the user can upload a client’s credit check report document.

 

Reference

Control

Functionality description

1

Modal header

Modal header titled “Upload Credit Check Report”.

2

Client label

Label titled “Client”.

3

Client textbox

Input of type Text with the clients Name and Surname. This is disabled as its populated with the clients Name and Surname.

4

Credit Check Report Document (.PDF) label

Label titled “Credit Check Report Document (.PDF”.

5

Choose file input

Input of type file to choose the document to upload. Click this to upload the credit check report document.

6

File name label

Label titled with the name of the document uploaded.

7

Document Type label

Label titled “Document Type”.

8

Document Type textbox

Input of type Text with the Document Type Description. This is disabled as its populated with the Document Type Description.

9

Save button

A button that will confirm the specific document to be uploaded by opening the “Upload this document?” Confirmation modal and closing the “Upload Credit Check Report” modal when clicked.

10

Cancel button

A button that closes the modal when clicked.
